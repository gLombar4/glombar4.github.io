Getting Started

What IS Java?

Java is an "Object Oriented" programming language used to design programs & apps. You'll learn what "Object-Oriented" means later, but for now just know that there are non-object-oriented languages as well.

So How Does it Work?

Java syntax will take a while to get used to, but here are the basics.

Java Essentials:

  • Class name & file name must match

  • There must be a "main" method (don't worry about all the stuff in it for now)
  • To get Java to actually show what you're programming, you'll need to output it via the "System.out.print()" command. This will output whatever you specify into the console.
  • Lastly, Java syntax operates via brackets: { }.
    You could technically write an entire program on just one line with appropriate brackets.
    (Please don't do that.)

And that's pretty much all you need just for your program to run. Java goes much deeper than this, however.
